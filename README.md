### Installing Packages

#### Linux/OSX/BSD Users

In your favorite $shell

```sh
$ cabal update
$ cabal install <package name>
```

#### Windows Users

Press `Windows+R` and type `cmd.exe` and then 
    
```sh
C:\> cabal update
C:\> cabal install <package name>
```
