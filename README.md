Installing Packages
-------------------

*Linux/OSX/BSD Users*

In your favorite $shell

```bash
$ cabal update
$ cabal install $packagename
```

*Windows Users*

Press `Windows+R` and type `cmd.exe` and then 
    
```cmd
C:\> cabal update
C:\> cabal install <package name>
```
